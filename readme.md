🚀 API YT Videos Project

🔗 Live Demo: https://api-yt-videos.vercel.app/

📌 Project Overview

A dynamic video listing application that fetches and displays YouTube-style video data using a public API.
This project focuses on API integration, data handling, and rendering dynamic UI components in a clean and responsive layout.

It simulates a simplified YouTube feed experience while maintaining performance, readability, and structured code practices.

✨ Why This Project

Working with external APIs
Handling nested JSON data
Rendering dynamic UI using JavaScript

Building clean and reusable UI structures
Creating responsive layouts

⚙️ How It Works
The app sends a request to a public YouTube-like API
Receives structured JSON data
Extracts nested properties like:
snippet (title, channel, thumbnail)
statistics (views)
Dynamically generates video cards using JavaScript
Renders them into the DOM
